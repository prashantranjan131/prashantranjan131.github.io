# Data Engineer & SQL Expert Portfolio

A modern, responsive personal portfolio website for a freelance Data Engineer and SQL Expert. Built to generate leads and attract clients who need help with SQL queries, PostgreSQL optimization, data cleaning, and debugging.

**[🌐 Live Website](https://prashantranjan131.github.io)**

---

## Overview

This portfolio showcases expertise in:
- **SQL Query Optimization** – Improving performance and reducing execution time
- **PostgreSQL Debugging** – Fixing errors, joins, and complex query issues
- **Data Cleaning & Transformation** – Structuring raw data for better insights
- **Data Engineering Support** – ETL pipelines and database tasks

## Getting Started

### Local Development

Simply open `index.html` in any modern web browser:

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
Or serve it locally for better testing:
# Using Python 3
python -m http.server 8000

# Using Node.js (npx)
npx serve .
Then visit http://localhost:8000.

Deployment

Go to Settings → Pages
Select Deploy from a branch → main
Your site will be live at https://prashantranjan131.github.io

Customization Checklist
Before going live, update the following in index.html:
[ ] Page Title & Meta – Add your name in <title> and <meta author>
[ ] Logo / Branding – Update initials and brand name in the navbar
[ ] Hero Section – Customize headline and subheading if needed
[ ] Case Studies – Replace example metrics with your real project results
[ ] About Me – Add your actual bio, experience, and location
[ ] Contact Email – Update mailto: link
[ ] WhatsApp Number – Update wa.me/ link with your real number
[ ] Availability Hours – Set your timezone and working hours
[ ] Footer – Update copyright and brand name
